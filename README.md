## Using the CLI (Commands)

> **Note** First fork the repo before using it.

- **To install it globally.**

```bash
npm install -g linkfree-cli

linkfree-Cli
```
It will then prompt you to questions.

- **To execute the package directly, without installing it.**

```bash
npx linkfree-cli
```
It will then prompt you to questions.
